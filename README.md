# Diwali Sales Analysis

## 📌 Overview
This project analyzes Diwali sales data to derive valuable insights into customer purchasing behavior, product performance, and sales trends. The goal is to leverage data-driven insights to optimize marketing strategies, enhance customer engagement, and maximize revenue.

## 🔍 Features
- **Data Cleaning & Preprocessing**: Handling missing values, formatting data, and preparing it for analysis.
- **Exploratory Data Analysis (EDA)**: Uncovering key insights through visualizations and descriptive statistics.
- **Customer Segmentation**: Analyzing demographics and spending patterns.
- **Sales Trends Analysis**: Identifying peak sales periods and best-selling categories.
- **Geographical Insights**: Mapping sales data by region.
- **Data Visualization**: Using Matplotlib and Seaborn for interactive and informative visuals.

## 📂 Dataset
The dataset contains transactional sales data from a Diwali festival, including:
- Customer details (Gender, Age, City, etc.)
- Product information (Category, Sub-category, Price, etc.)
- Purchase behavior (Quantity, Total Amount, Payment Mode, etc.)

## 🛠️ Tech Stack
- **Python**
- **Pandas & NumPy** (Data manipulation & analysis)
- **Matplotlib & Seaborn** (Data visualization)
- **Jupyter Notebook** (Development environment)

## 📊 Key Insights
- **Top Performing Product Categories**: Electronics and fashion-related products had the highest sales.
- **Peak Sales Periods**: Most purchases occurred in the days leading up to Diwali.
- **Customer Behavior**: Male customers had higher purchase values, and metropolitan cities contributed the most revenue.
